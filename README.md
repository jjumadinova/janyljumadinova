# Janyl Jumadinova's Website

Source code for [Janyl Jumadinova's website](https://janyljumadinova.com).

It is based on [huxblog Jekyll Theme](https://github.com/Huxpro/huxpro.github.io), [Clean Blog Jekyll Theme](https://github.com/BlackrockDigital/startbootstrap-clean-blog-jekyll), and [Clean White Theme for Hugo](https://github.com/zhaohuabing/hugo-theme-cleanwhite).

## Installing and Running the Website

You can clone the repository for the website by running the following command:

```
$ git clone https://github.com/jjumadinova/janyljumadinova.git
```

Then, you can run the website with build-in `hugo` server locally:

```
$ hugo serve
```

[`localhost:1313`](http://localhost:1313) will display the website.
